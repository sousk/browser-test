Audio Element & Web Audio API Tester

  • Play HE-AAC
  • Play HE-AAC (Audio Element -> GainNode)
  • Play MP3
  • AudioElement: Play HE-AAC
  • At Safari, audio element cannot be connected with Web Audio API even through createMediaElementSource, but Chrome does.
  • You have to control audio element to play sound. Web Audio API just works as a filter so that you cannot play if audio element is pused or muted.
  • Note that:
    • Audio Element requires explicit user action to start playing both on Mobile Safari and Chrome.
    • Web Audio API requires explicit user action only on Mobile Safari.
